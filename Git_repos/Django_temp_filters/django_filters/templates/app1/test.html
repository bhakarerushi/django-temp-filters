Welcome to Django-Filter Page

Django provides multiple filters to use in templares.

django template filters:
  • add filter for addition on variable : {{value | add:"2" }}
  • capfirst filter to make first letter capital : {{ backend | capfirst }}
  • center filter to make it center for specified range : {{ backend | center:"15"}}
  • cut filter removes arg from given string : {{ data | cut:" "}}
  • date filter place date in specified format : {{ date | date:"d M Y" }}
  • first filter return first ele of given list : {{ list | first}}
  • floatformat filter round float upto args : {{val | floatformat:"-2" }}
  • ger_digit filter specified no. digit from right :{{ val | get_digit:"2" }}
  • join filter joins list with specified arg. : {{ list| join:"+"}}
  • length_is filter return true if specified length is equal to variable len :{{ list | length_is:"4"}}
  • random filter return random item from list : {{ list | random }}
